# CI/CD Fullstack Task Manager

This is a full-stack DevOps project featuring a Spring Boot Java backend and a React (Vite) frontend. The application is fully containerized using Docker and utilizes a CI/CD pipeline for automated deployments.

## Tech Stack
* **Frontend:** React, Vite, CSS
* **Backend:** Java, Spring Boot, H2 In-Memory Database
* **DevOps:** Docker, Docker Compose, GitHub Actions, Render, Netlify
